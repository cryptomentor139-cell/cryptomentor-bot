🚀 PROMPT REPLIT AI AGENT — Perbaiki Admin Gate (pakai ADMIN, ADMINS, dll.)

GOAL

Gunakan Secret ADMIN sebagai sumber ID admin (bisa satu atau banyak, dipisah koma/spasi/newline).

Tetap mendukung fallback: ADMINS, ADMIN_IDS, ADMIN_ID, ADMIN1..ADMIN10.

Perbaiki semua admin-gated features (termasuk /admin) untuk memakai fungsi pusat is_admin(user_id).

Perbarui pesan “Admin Access Required” agar menampilkan daftar admin aktual (bukan NONE CONFIGURED bila sudah ada).

Jangan mengubah fitur lain; kirim /admin tetap HTML+chunking & waktu UTC seperti sekarang.

1) Buat modul pusat otorisasi: app/auth.py

Tambahkan file ini dan jadikan satu-satunya sumber kebenaran admin list.

# app/auth.py
import os
import re
from typing import List, Set

# Kunci env yang didukung (urutan prioritas):
ENV_KEYS = [
    "ADMIN",          # ← utama (bisa 1 atau banyak)
    "ADMINS",
    "ADMIN_IDS",
    "ADMIN_ID",
]
# Dukungan legacy: ADMIN1..ADMIN10
LEGACY_KEYS = [f"ADMIN{i}" for i in range(1, 11)]

def _split_candidates(val: str) -> List[str]:
    """Pisah string berdasarkan koma, spasi, newline, titik koma."""
    # Hapus bracket/quotes bila ada
    val = val.strip().strip("[](){}\"'")
    # Gantikan pemisah umum jadi spasi
    val = re.sub(r"[,;\n\r\t]+", " ", val)
    parts = [p for p in val.split(" ") if p]
    return parts

def _coerce_id(s: str):
    s = s.strip()
    m = re.search(r"\d+", s)
    if not m:
        return None
    try:
        return int(m.group(0))
    except Exception:
        return None

def get_admin_ids() -> List[int]:
    ids: Set[int] = set()

    # 1) baca ENV_KEYS dahulu
    for k in ENV_KEYS:
        v = os.getenv(k)
        if v:
            for p in _split_candidates(v):
                uid = _coerce_id(p)
                if uid is not None:
                    ids.add(uid)

    # 2) legacy ADMIN1..ADMIN10
    for k in LEGACY_KEYS:
        v = os.getenv(k)
        if v:
            uid = _coerce_id(v)
            if uid is not None:
                ids.add(uid)

    # Hasil terurut
    return sorted(ids)

def is_admin(user_id: int) -> bool:
    try:
        uid = int(user_id)
    except Exception:
        return False
    return uid in set(get_admin_ids())

def admin_denied_text(requester_id: int) -> str:
    admins = get_admin_ids()
    admins_text = ", ".join(map(str, admins)) if admins else "NONE CONFIGURED"
    return (
        "❌ Admin Access Required\n\n"
        f"Your ID: {requester_id}\n"
        f"Admin IDs: {admins_text}\n\n"
        "⚙️ Setup Instructions:\n"
        "1) Buka Secrets tab di Replit\n"
        "2) Tambahkan salah satu:\n"
        "   • ADMIN = 7255533151            (satu ID)\n"
        "   • ADMIN = 7255533151, 123456789 (multi ID, pisah koma)\n"
        "   • ADMINS / ADMIN_IDS juga boleh\n"
        "   • Legacy: ADMIN1, ADMIN2, ...\n"
        "3) Restart bot untuk apply changes\n"
    )

2) Patch handler /admin supaya pakai is_admin

aiogram v3 atau telebot — terapkan yang sesuai proyekmu.

a) Jika aiogram v3 (app/routers/admin_router.py)

Ganti handler /admin seperti ini (bagian escape HTML + chunking tetap):

# app/routers/admin_router.py
from aiogram import Router, types
from app.admin import get_admin_panel_text
from app.auth import is_admin, admin_denied_text
import html

router = Router()

def _to_html_chunks(text: str, max_len: int = 3500):
    esc = html.escape(text, quote=False)
    buf, cur = [], 0
    for line in esc.splitlines(True):
        if cur + len(line) > max_len and buf:
            yield "".join(buf)
            buf, cur = [line], len(line)
        else:
            buf.append(line); cur += len(line)
    if buf:
        yield "".join(buf)

@router.message(commands={"admin"})
async def admin_panel(msg: types.Message):
    uid = msg.from_user.id if msg.from_user else 0
    if not is_admin(uid):
        for part in _to_html_chunks(admin_denied_text(uid)):
            await msg.answer(part, parse_mode="HTML", disable_web_page_preview=True)
        return

    text = get_admin_panel_text()  # sudah UTC
    for part in _to_html_chunks(text):
        await msg.answer(part, parse_mode="HTML", disable_web_page_preview=True)

4) Acceptance

Set Secret ADMIN = 7255533151 di Replit (atau beberapa ID dipisah koma).

Restart, lalu kirim /admin.

Tidak lagi muncul “Admin IDs: NONE CONFIGURED”.

Panel terbuka untuk ID yang terdaftar di ADMIN.

Jika mencoba dari akun lain (tidak ada di ADMIN), muncul pesan denied dengan daftar Admin IDs yang benar.

Catatan cepat

Parser menerima format fleksibel:
ADMIN="7255533151, 123456" atau ADMINS="7255533151 123456" atau ADMIN1=7255533151, dst.

Kalau mau load dari Supabase di masa depan, cukup ubah get_admin_ids() untuk men-query tabel admin — semua handler lain tak perlu diubah karena sudah terpusat.

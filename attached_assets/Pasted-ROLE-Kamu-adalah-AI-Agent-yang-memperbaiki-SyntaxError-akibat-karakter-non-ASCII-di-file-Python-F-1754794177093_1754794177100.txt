ROLE
Kamu adalah AI Agent yang memperbaiki SyntaxError akibat karakter non-ASCII di file Python.
Fokus pada file: /home/runner/workspace/Bismillah/bot.py
Error terkini: line 2237 → karakter '✅' (U+2705) di string f"""..."""

GOAL
1) Hilangkan SEMUA karakter non-ASCII (emoji/simbol) dari source Python.
2) Ganti dengan padanan ASCII aman, tanpa mengubah logika/ekspresi Python.
3) Pastikan file dapat di-import tanpa SyntaxError.

SCOPE
- Perbaiki line 2237: ganti "✅" jadi "[OK]" dan pastikan triple-quoted string aman.
- Scan seluruh file .py untuk karakter non-ASCII lain dan bersihkan konsisten (bullet, emoji, dash melengkung, kutip melengkung, dll).

MANDATORY RULES (ASCII-ONLY)
- DILARANG menyisakan karakter non-ASCII (codepoint > 127) di source.
- Mapping penggantian standar:
  - ✅, ✔, ✓ → "[OK]"
  - ❌, ✖ → "[ERR]"
  - ⚠ → "[WARN]"
  - 🚀 → "[START]"
  - 🔧 → "[TOOL]"
  - 🔍 → "[SCAN]"
  - 📊 → "[STAT]"
  - 📅 → "[DATE]"
  - • (U+2022) → "-"
  - – (U+2013), — (U+2014) → "-"
  - “ ” ‘ ’ → " (double quote) / ' (single quote) ASCII
- Jika ada non-ASCII lain yang tidak terpetakan, hapus atau ganti dengan padanan ASCII terdekat.
- JANGAN ubah ekspresi Python di dalam f-string; hanya teks literalnya.
- Tambahkan header encoding jika belum ada:
  # -*- coding: utf-8 -*-

STRING/DOC RULES
- Gunakan ASCII saja dalam string/docstring.
- Untuk blok panjang, boleh pakai triple quotes, tapi tanpa emoji dan tanpa indentasi yang aneh; atau bangun string pakai '\n'.

STEPS
1) Buka /home/runner/workspace/Bismillah/bot.py.

PROMPT — Fix ImportError rate_limiter + Safe Rate Limit Utilities
1) Config flag

Edit app/config.py dan tambahkan:

import os
USE_RATE_LIMIT = os.getenv("USE_RATE_LIMIT", "true").lower() in ("1", "true", "yes")
RATE_LIMIT_QPS = float(os.getenv("RATE_LIMIT_QPS", "1.5"))  # ~1–2 req/detik per user
BURST_TOKENS = int(os.getenv("BURST_TOKENS", "3"))          # burst kecil


Kamu bisa matikan limiter sementara dengan USE_RATE_LIMIT=false di Secrets—bot tetap jalan.

2) Modul limiter yang mengekspor rate_limiter

Buat/replace app/utils/rate_limiter.py:

from __future__ import annotations
import time
import asyncio
from typing import Dict, Tuple, Callable, Awaitable, Optional, Hashable
from app.config import USE_RATE_LIMIT, RATE_LIMIT_QPS, BURST_TOKENS

class RateLimiter:
    """
    Token-bucket per key (mis. per-user atau per-command).
    - qps: token per detik (rata-rata)
    - burst: jumlah token maksimum (burst allowance)
    - key: (scope, user_id) tuple, atau string lain yang hashable
    """
    def __init__(self, qps: float = 1.0, burst: int = 3):
        self.qps = max(0.01, qps)
        self.burst = max(1, burst)
        self._store: Dict[Hashable, Tuple[float, float]] = {}  # key -> (tokens, last_ts)
        self._lock = asyncio.Lock()

    def _refill(self, tokens: float, last_ts: float) -> Tuple[float, float]:
        now = time.monotonic()
        tokens = min(self.burst, tokens + (now - last_ts) * self.qps)
        return tokens, now

    async def allow(self, key: Hashable, cost: float = 1.0) -> bool:
        if not USE_RATE_LIMIT:
            return True
        async with self._lock:
            tokens, last_ts = self._store.get(key, (self.burst, time.monotonic()))
            tokens, now = self._refill(tokens, last_ts)
            if tokens >= cost:
                tokens -= cost
                self._store[key] = (tokens, now)
                return True
            # belum cukup token
            self._store[key] = (tokens, now)
            return False

    async def wait(self, key: Hashable, cost: float = 1.0) -> None:
        """Block until allowed (gunakan untuk internal jobs)."""
        if not USE_RATE_LIMIT:
            return
        while True:
            if await self.allow(key, cost):
                return
            # hitung waktu tunggu kira2
            await asyncio.sleep(max(0.05, cost / self.qps))

# ---- Instance global yang diharapkan ada oleh kode lain ----
rate_limiter = RateLimiter(qps=RATE_LIMIT_QPS, burst=BURST_TOKENS)

# ---- Decorator helper untuk handlers ----
def limit_command(scope: str, cost: float = 1.0):
    """
    Pakai di handler Telegram:
    @limit_command("analyze", cost=1)
    async def cmd_analyze(update, context): ...
    """
    def wrap(fn: Callable[..., Awaitable]):
        async def inner(*args, **kwargs):
            # cari update untuk ambil user.id
            update = None
            if args:
                for a in args:
                    # PTB: update arg pertama/ kedua
                    if getattr(a, "effective_user", None):
                        update = a
                        break
            if update and getattr(update, "effective_user", None):
                uid = str(update.effective_user.id)
                key = (scope, uid)
            else:
                key = (scope, "anonymous")

            if await rate_limiter.allow(key, cost=cost):
                return await fn(*args, **kwargs)
            else:
                # throttle message minimalis; jangan raise agar tidak crash
                try:
                    msg = getattr(update, "effective_message", None)
                    if msg:
                        await msg.reply_text("⏳ Terlalu banyak permintaan. Coba lagi sebentar...")
                except Exception:
                    pass
                return None
        return inner
    return wrap


PENTING: Modul ini menyediakan rate_limiter sehingga from app.utils.rate_limiter import rate_limiter tidak error lagi.

3) Perbaiki semua import yang lama

Cari dan ganti impor yang menyebabkan crash:

Dulu (crash):

from app.utils.rate_limiter import rate_limiter  # OK, ini yang kita jaga


atau

from app.utils import rate_limiter  # <-- ini salah kalau modul berharap symbol


Sekarang (aman):

from app.utils.rate_limiter import rate_limiter, limit_command

4) Tambahkan limiter ringan ke handlers (opsional tapi disarankan)

Contoh pada empat command inti:

Edit app/commands/handlers.py:

from app.utils.rate_limiter import limit_command

@limit_command("analyze", cost=1)
async def cmd_analyze(update, context):
    # ... isi tetap ...

@limit_command("market", cost=1)
async def cmd_market(update, context):
    # ... isi tetap ...

    # ... isi tetap ...

@limit_command("futures_signals", cost=1)
async def cmd_futures_signals(update, context):
    # ... isi tetap ...


Jika kamu tidak ingin limiter aktif sekarang, cukup set USE_RATE_LIMIT=false di Secrets. Importnya tetap aman dan tidak akan crash.

5) (Opsional) Guard aman di tempat lain

Jika ada kode yang memanggil rate_limiter.allow(...) secara manual, biarkan—instance sudah tersedia.
Kalau ada middleware lama yang impor rate_limiter dari path berbeda, samakan semuanya ke:

from app.utils.rate_limiter import rate_limiter

6) Quick test (tanpa menjalankan bot penuh)

Buat file kecil scripts/rl_test.py:

import asyncio
from app.utils.rate_limiter import rate_limiter

async def main():
    key=("test","123")
    allowed=0
    for _ in range(10):
        if await rate_limiter.allow(key): allowed+=1
        await asyncio.sleep(0.1)
    print("allowed:", allowed)

asyncio.run(main())


Jalankan:

python scripts/rl_test.py


Kamu harus melihat angka “allowed” yang masuk akal (tergantung QPS/BURST).

7) Tentang “⚠️ Cleanup error: This Application is not running!”

Itu cuma efek samping dari crash sebelumnya (loop aplikasi belum sempat start).
Begitu ImportError di atas beres, pesan itu akan hilang saat bot berhasil start normal.

🚀 PROMPT REPLIT AI AGENT — Fix ModuleNotFoundError: aiogram

GOAL

Pasang aiogram v3 (cocok dengan kode Router/Middleware yang kita pakai).

Pastikan python main.py tidak error import lagi.

Jangan ubah logic bot selain bootstrap.

TASKS

Buat/Update requirements.txt di root proyek berisi (tambahkan baris jika belum ada):

aiogram==3.4.1
httpx>=0.24
supabase>=2.4
python-dotenv>=1.0


Tambahkan bootstrap failsafe agar kalau lingkungan belum install, skrip akan meng-install di runtime lalu lanjut jalan.

Buat file app/bootstrap_deps.py:

# app/bootstrap_deps.py
import importlib, subprocess, sys

REQS = [
    ("aiogram", "aiogram==3.4.1"),
    ("httpx", "httpx>=0.24"),
]

def ensure():
    for mod, pip_spec in REQS:
        try:
            importlib.import_module(mod)
        except Exception:
            subprocess.check_call([sys.executable, "-m", "pip", "install", pip_spec])

if __name__ == "__main__":
    ensure()


Patch main.py untuk memanggil bootstrap sebelum import aiogram.

Di paling atas sebelum from aiogram import ..., tambahkan:

# main.py (baris teratas)
try:
    from app.bootstrap_deps import ensure as _ensure_deps
except Exception:
    def _ensure_deps(): pass
_ensure_deps()

from aiogram import Bot, Dispatcher
# (lanjutan import & kode kamu)


(Opsional tapi dianjurkan) Pastikan token bot ada supaya tidak crash setelah import:

Di main.py, ambil token seperti ini:

import os, sys, asyncio
from aiogram.enums import ParseMode

TOKEN = os.getenv("BOT_TOKEN") or os.getenv("TELEGRAM_BOT_TOKEN")
if not TOKEN:
    print("ERROR: Set BOT_TOKEN di Secrets Replit.", file=sys.stderr)
    sys.exit(1)

async def main():
    bot = Bot(TOKEN, parse_mode=ParseMode.HTML)
    dp = Dispatcher()
    # include routers kamu di sini...
    await dp.start_polling(bot)

if __name__ == "__main__":
    asyncio.run(main())


Jalankan ulang Replit.

Replit akan membaca requirements.txt dan meng-install; bila belum, bootstrap akan meng-install di runtime sehingga import aiogram sukses.

ACCEPTANCE

Menjalankan proyek tidak lagi error:

ModuleNotFoundError: No module named 'aiogram'


Bot start, polling aktif.

Command sebelumnya (/admin, dll.) tetap berjalan.

Catatan:

Kita pakai aiogram v3 (sesuai semua patch Router/Middleware yang kamu punya).

Kalau kamu masih punya kode lama bergaya v2 (mis. Dispatcher(bot) lama, atau executor.start_polling), konsolidasikan ke v3 seperti kerangka di atas.

Kalau Replit-mu pakai Poetry/pyproject, Agent boleh poetry add aiogram==3.4.1 alih-alih requirements.txt.

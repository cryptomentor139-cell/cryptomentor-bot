PROMPT — REVERT /futures_signals KE VERSI LAMA
1) Tambah feature flag di config

Edit app/config.py:

import os

# ...
USE_LEGACY_FUTURES_SIGNALS = os.getenv("USE_LEGACY_FUTURES_SIGNALS", "true").lower() in ("1","true","yes")


Default aku set ke true biar langsung pakai versi lama. Kamu bisa ubah di Secrets:
USE_LEGACY_FUTURES_SIGNALS=true/false.

2) Buat service versi lama

Buat file app/services/futures_legacy.py:

from typing import List, Dict, Any
from app.utils.async_tools import gather_safe  # pastikan util ini sudah ada
from app.services.analysis import analyze_coin_futures  # gunakan analyzer yang sekarang (entry tunggal tetap)

DEFAULT_COINS = ["BTC", "ETH", "BNB", "SOL", "XRP"]  # list lama standar, sesuaikan jika perlu

async def futures_signals_legacy(coins: List[str] | None) -> List[Dict[str, Any]]:
    """
    Versi lama:
    - Tidak scan Top 30 otomatis.
    - Tidak ada filter confidence.
    - Jika argumen kosong, gunakan DEFAULT_COINS.
    - Kembalikan list dict apa adanya & biarkan formatter lama yang render.
    """
    coins_up = [c.upper() for c in (coins or DEFAULT_COINS)]
    tasks = [analyze_coin_futures(c) for c in coins_up]
    results = await gather_safe(tasks)

    out: List[Dict[str, Any]] = []
    for c, r in zip(coins_up, results):
        if isinstance(r, Exception):
            out.append({"coin": c, "error": str(r)})
        else:
            # Hapus field "confidence"/"ok" jika versi lama tidak memakainya
            r.pop("confidence", None)
            r.pop("ok", None)
            out.append(r)
    return out


Catatan: Kita tetap pakai engine analisis terbaru (biar /futures tetap 1 titik entry), tapi hasilnya tidak disaring lagi.

3) Wire-up di handler agar balik ke legacy saat flag ON

Edit app/commands/handlers.py (bagian /futures_signals):

import importlib
from telegram import Update
from telegram.ext import ContextTypes
from app.config import USE_LEGACY_FUTURES_SIGNALS

# Import keduanya: legacy & modern
from app.services.futures_legacy import futures_signals_legacy
from app.services.analysis import futures_signals as futures_signals_modern

async def cmd_futures_signals(update: Update, context: ContextTypes.DEFAULT_TYPE):
    coins = [c.upper() for c in (context.args or [])]
    try:
        if USE_LEGACY_FUTURES_SIGNALS:
            lst = await futures_signals_legacy(coins or None)
        else:
            # modern (Top30 + threshold) jika suatu saat kamu butuh lagi
            lst = await futures_signals_modern(coins or None, threshold=75.0)

        # Coba formatter lama dulu biar output identik
        F = None
        try:
            F = importlib.import_module("app.formatters.texts")
        except Exception:
            pass

        if F and hasattr(F, "format_futures_signals"):
            text = F.format_futures_signals(lst)
        else:
            # Fallback simple, menyerupai lama
            lines = ["🚨 FUTURES SIGNALS"]
            if not lst:
                lines += ["❌ Tidak ada sinyal"]
            else:
                for it in lst:
                    if "error" in it:
                        lines.append(f"- {it['coin']}: ❌ error")
                        continue
                    lines.append(
                        f"- {it['coin']}: Entry {it['entry']:.4f} | "
                        f"RSI {it['rsi']:.1f} | MACD {it['macd_hist']:.4f} | {it['trend']}"
                    )
            text = "\n".join(lines)

        await update.effective_message.reply_text(text)
    except Exception as e:
        await update.effective_message.reply_text(
            "❌ **Terjadi kesalahan mengambil data untuk FUTURES_SIGNALS**\n\n"
            "🔄 **Error context**: legacy mode\n"
            f"`{e}`\n"
            "💡 Coba: /price btc — Cek harga basic\n",
            parse_mode="Markdown",
        )

4) (Opsional) Pastikan util gather ada

Kalau kamu belum punya utilnya, buat app/utils/async_tools.py:

import asyncio
from typing import Iterable, Awaitable, List, Any

async def gather_safe(tasks: Iterable[Awaitable[Any]]) -> List[Any]:
    return await asyncio.gather(*tasks, return_exceptions=True)

Tugasmu adalah memperbaiki seluruh integrasi CoinGlass V4 API agar bisa digunakan dalam fitur seperti `/analyze`, `/futures`, dan auto signals secara real-time.

Berikut arahan detail yang harus kamu ikuti:

---

🔧 1. Perbaiki Symbol Mapping
• Gunakan symbol standar seperti `BTCUSDT`, `ETHUSDT`, `SANDUSDT` apa adanya.
• Jangan ubah menjadi `BTCTUSDT` atau `SANDTUSDT`. Itu penyebab utama error 404.
• Tambahkan validasi symbol berdasarkan `/futures/supported-coins` atau `/futures/supported-exchange-pairs`.

---

📡 2. Gunakan Endpoint CoinGlass V4 Resmi
Gantilah seluruh endpoint dummy atau yang keliru menjadi endpoint berikut:

### ✅ Open Interest
`GET /api/futures/openInterest/exchange-list`

### ✅ Funding Rate
`GET /api/futures/fundingRate/exchange-list`

### ✅ Long/Short Ratio
`GET /api/futures/top-long-short-position-ratio/history`
atau
`GET /api/futures/top-long-short-account-ratio/history`

### ✅ Liquidation Map
`GET /api/futures/liquidation/map` _(pair-level)_
`GET /api/futures/liquidation/aggregated-map` _(coin-level)_

### ✅ Price OHLC or Market Price
`GET /api/spot/price/history` atau `GET /futures/price-change-list`

---

🚨 3. Tangani Error 404
• Jangan gunakan harga dummy seperti `$70,000.00` bila data gagal.
• Jika CoinGlass mengembalikan 404, tampilkan pesan:
  “⚠️ Data tidak tersedia untuk pair ini di CoinGlass.”
• Tambahkan log internal tapi jangan crash-kan sistem.

---

🎨 4. Hindari Telegram Markdown Error
• Escape karakter seperti `.` atau `-` dalam mode `MarkdownV2`.
• Atau gunakan `parse_mode='HTML'` sebagai default fallback jika parsing gagal.

---

🧠 5. Perbaiki Command Terkait:
• `/analyze <symbol>`
• `/futures <symbol>`
• `/market`
• `/futures_signals`
• Auto SND Scanner
Pastikan semua command ini:
✅ Mengambil data CoinGlass asli  
✅ Tidak menggunakan dummy  
✅ Tidak error parsing  
✅ Support coin seperti BTC, ETH, SAND, MATIC, LINK

---

🎯 6. Uji Hasil Final
Setelah selesai:
• Jalankan `/futures btc`, `/analyze eth`, dan `/market`
• Pastikan semua data muncul akurat dan tidak dummy
• Tidak ada error 404, tidak ada simbol salah

---

🎁 Bonus
Tambahkan fitur fallback yang menggunakan CoinAPI atau CoinMarketCap **hanya jika** CoinGlass gagal.

Tujuan utama: Semua data **futures real-time CoinGlass V4** berhasil digunakan di semua fitur.

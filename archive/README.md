# Archive Folder Structure

This folder contains organized historical files from the project.

## Folder Structure

### 📁 docs/
Documentation files including reports, summaries, analyses, and guides.

### 📁 tests/
Test scripts and verification tools.

### 📁 deploy/
Deployment scripts and related files.
- `docs/` - Deployment documentation
- `commands/` - Command reference files

### 📁 specs/
Specification and design documents.

### 📁 fixes/
Bug fix documentation and issue reports.

### 📁 features/
Feature-specific documentation organized by feature name:
- StackMentor system
- Scalping mode
- Risk management
- Social proof
- BingX integration
- Community features
- License system
- Whitelabel
- AutoTrade engine
- And more...

### 📁 middleware/
Middleware and integration documentation.

### 📁 misc/
Miscellaneous scripts and utilities.

## Note

These files are archived for reference. Active development files are in the main project folders:
- `Bismillah/` - Main bot code
- `db/` - Database migrations
- `.kiro/specs/` - Active specifications
- `license_server/` - License server
- `Whitelabel #1/` - Whitelabel instance
